# PalindromeCheckerApp

## Overview

This repository contains a Java application that checks whether a given text input is a palindrome using three different approaches:

- Two-pointer comparison
- Stack comparison
- Linked list comparison


## Requirements

- Java Development Kit (JDK) 8 or later

## Usage

When the program runs, it prompts for input text and prints:

- the entered text
- whether the text is a palindrome for each approach
- execution time for each approach in nanoseconds
